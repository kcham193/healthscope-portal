1. Background

The Kenya Master Health Facility Registry (KMHFR) at kmhfr.health.go.ke is operated by the Kenya Ministry of Health. It contains the authoritative registry of all health facilities in Kenya, including their type, ownership, KEPH level, county, GPS coordinates, and services offered.

The KMHFR does not provide a public API. The platform is built with Next.js (React Server Components) and renders facility data server-side, embedding it in the HTML as escaped JSON within self.__next_f.push() script blocks. This means data cannot be accessed via simple REST API calls — it must be extracted from the rendered HTML pages.

2. Extraction Approach

The extraction is performed in three phases:

1 Facility Listing Scrape

The facility listing at /public/facilities?page=N displays 30 facilities per page. With ~17,000+ facilities, this means ~580 pages. Each page's HTML contains a self.__next_f.push([1,"..."]) block with embedded JSON facility objects.

Data extracted per facility:

  • id — UUID (used to access the detail page)
  • name, official_name, code
  • facility_type_name (e.g., Dispensary, Hospital, Medical Clinic)
  • owner_name (e.g., Ministry of Health, Private Practice)
  • operation_status_name (Operational, Closed, etc.)
  • county_name, sub_county_name, constituency_name, ward_name
  • keph_level_name (Level 2 through Level 6)

Concurrency: 8 threads • Time: ~5 minutes

2 Detail Page Scrape (Coordinates + Services)

Each facility has a detail page at /public/facilities/{uuid} containing GPS coordinates and the full list of services. The detail page HTML includes:

  • Coordinates: Embedded as "lat_long":[-1.30088,36.80734] in a JSON object within the push blocks
  • Services: Each service appears as a "service_name":"..." paired with a "category_name":"..." in the push blocks

The services tab on each facility page shows entries like:

3 Standardization & Export

Raw data is standardized into two CSV files matching the HealthScope schema:

  • kenya_standardized.csv — One row per facility (name, code, type, ownership, lat, lon, county, status)
  • kenya_services.csv — Long format, one row per service per facility (facility_code, service_category, service_group, service_detail)

Coordinates are validated against Kenya's bounding box (lat: -4.7 to 5.5, lon: 33.9 to 41.9). Ownership is classified into Public, Private, Faith-Based, or NGO.

3. Key Extraction Patterns

3.1 Facility Objects from Listing Pages

Facility data is embedded in self.__next_f.push([1,"..."]) blocks as escaped JSON. After unescaping \" → ", we match:

{"id":"[a-f0-9-]{36}",
 "facility_type_name":"...",
 "owner_name":"...",
 "operation_status_name":"...",
 "constituency_name":"...",
 "ward_name":"...",
 "sub_county_name":"...",
 "county_name":"...",
 "keph_level_name":"...",
 "name":"...",
 "official_name":"...",
 "code":12345}

3.2 Coordinates from Detail Pages

# Regex pattern:
re.search(r'lat_long..:\[([0-9.\-]+),([0-9.\-]+)\]', html)

# Example match in HTML:
"lat_long":[-1.30088,36.80734]

3.3 Services from Detail Pages

# After unescaping push block content:
service_names = re.findall(r'"service_name":"([^"]{3,120})"', text)
category_names = re.findall(r'"category_name":"([^"]{3,120})"', text)

# These appear in matched pairs (same order), so:
for name, category in zip(service_names, category_names):
    services.append({"name": name, "category": category})

5. Full Extraction Script

Requirements: Python 3.10+ (standard library only — no pip packages needed).
Runtime: ~40-60 minutes depending on network speed.
Usage: python kenya_extraction.py 
#!/usr/bin/env python3
"""
Kenya Master Health Facility Registry (KMHFR) - Full Data Extraction
Author:  Kasim Chambulilo
Project: HealthScope Intelligence Portal
Source:  https://kmhfr.health.go.ke/public/facilities
"""

import json, csv, re, os, sys, ssl, time, urllib.request
from concurrent.futures import ThreadPoolExecutor, as_completed

# ── Configuration ──
OUTPUT_DIR = "output"
THREADS_LISTING = 8
THREADS_DETAIL = 10
TIMEOUT = 30
UA = "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36"
KENYA_BOUNDS = (-4.7, 5.5, 33.9, 41.9)  # lat_min, lat_max, lon_min, lon_max

CTX = ssl.create_default_context()
CTX.check_hostname = False
CTX.verify_mode = ssl.CERT_NONE


def fetch(url):
    req = urllib.request.Request(url, headers={"User-Agent": UA})
    return urllib.request.urlopen(req, timeout=TIMEOUT, context=CTX) \
               .read().decode("utf-8", "replace")


# ══════════════════════════════════════════════
#  PHASE 1: Scrape facility listing pages
# ══════════════════════════════════════════════

def extract_listings(html):
    """Extract facility objects from a listing page HTML."""
    pushes = re.findall(r'self\.__next_f\.push\(\[1,"(.*?)"\]\)', html, re.DOTALL)
    facs = []
    for p in pushes:
        content = p.replace('\\"', '"').replace('\\u0026', '&')
        for m in re.finditer(
            r'\{"id":"([a-f0-9-]{36})",'
            r'"facility_type_name":"([^"]*)",'
            r'"owner_name":"([^"]*)",'
            r'"operation_status_name":"([^"]*)",'
            r'"constituency_name":"([^"]*)",'
            r'"ward_name":"([^"]*)",'
            r'"sub_county_name":"([^"]*)",'
            r'"county_name":"([^"]*)",'
            r'"keph_level_name":"([^"]*)",'
            r'"name":"([^"]*)",'
            r'"official_name":"([^"]*)",'
            r'"code":(\d+|null)', content):
            facs.append({
                "id": m.group(1), "facility_type": m.group(2),
                "owner": m.group(3), "status": m.group(4),
                "county": m.group(8), "sub_county": m.group(7),
                "keph_level": m.group(9), "name": m.group(10),
                "code": m.group(12) if m.group(12) != "null" else "",
            })
    return facs


def fetch_listing_page(page):
    html = fetch(f"https://kmhfr.health.go.ke/public/facilities?page={page}")
    return page, extract_listings(html)


def phase1():
    print("PHASE 1: Scraping listing pages...", flush=True)
    all_facs = {}
    errors = 0
    total_pages = 600  # overshoot; empty pages are harmless
    with ThreadPoolExecutor(max_workers=THREADS_LISTING) as ex:
        futures = {ex.submit(fetch_listing_page, p): p
                   for p in range(1, total_pages + 1)}
        done = 0
        for f in as_completed(futures):
            done += 1
            try:
                _, facs = f.result()
                for fac in facs:
                    all_facs[fac["id"]] = fac
            except:
                errors += 1
            if done % 50 == 0:
                print(f"  {done}/{total_pages} pages, "
                      f"{len(all_facs)} facilities, {errors} errors",
                      flush=True)
    print(f"  Phase 1 done: {len(all_facs)} facilities\n", flush=True)
    return all_facs


# ══════════════════════════════════════════════
#  PHASE 2: Scrape detail pages (coords + services)
# ══════════════════════════════════════════════

def fetch_detail(fac):
    """Fetch one facility detail page, extract coords + services."""
    uid = fac["id"]
    html = fetch(f"https://kmhfr.health.go.ke/public/facilities/{uid}")

    # Coordinates
    lat, lon = "", ""
    m = re.search(r"lat_long..:\[([0-9.\-]+),([0-9.\-]+)\]", html)
    if m:
        lat, lon = m.group(1), m.group(2)

    # Services (from push blocks)
    pushes = re.findall(
        r'self\.__next_f\.push\(\[1,"(.*?)"\]\)', html, re.DOTALL)
    text = " ".join(p.replace('\\"', '"').replace("\\u0026", "&")
                    for p in pushes)
    svc_names = re.findall(r'"service_name":"([^"]{3,120})"', text)
    cat_names = re.findall(r'"category_name":"([^"]{3,120})"', text)

    svcs = []
    if len(svc_names) == len(cat_names):
        svcs = [{"name": n, "category": c}
                for n, c in zip(svc_names, cat_names)]
    else:
        svcs = [{"name": n, "category": ""} for n in svc_names]

    return uid, lat, lon, svcs


def phase2(all_facs):
    print(f"PHASE 2: Scraping {len(all_facs)} detail pages...", flush=True)
    coords, services = {}, {}
    errors = 0
    total = len(all_facs)
    with ThreadPoolExecutor(max_workers=THREADS_DETAIL) as ex:
        futures = {ex.submit(fetch_detail, f): f["id"]
                   for f in all_facs.values()}
        done = 0
        for f in as_completed(futures):
            done += 1
            try:
                uid, lat, lon, svcs = f.result()
                if lat and lon:
                    coords[uid] = (lat, lon)
                if svcs:
                    services[uid] = svcs
            except:
                errors += 1
            if done % 500 == 0:
                print(f"  {done}/{total}: {len(coords)} coords, "
                      f"{len(services)} w/svcs, {errors} errors",
                      flush=True)
    print(f"  Phase 2 done: {len(coords)} coords, "
          f"{len(services)} w/services\n", flush=True)
    return coords, services


# ══════════════════════════════════════════════
#  PHASE 3: Export standardized CSVs
# ══════════════════════════════════════════════

def phase3(all_facs, coords, services):
    print("PHASE 3: Exporting CSVs...", flush=True)
    os.makedirs(OUTPUT_DIR, exist_ok=True)
    b = KENYA_BOUNDS

    # Facility CSV
    rows = []
    for fac in all_facs.values():
        lat, lon = "", ""
        if fac["id"] in coords:
            la, lo = coords[fac["id"]]
            try:
                la_f, lo_f = float(la), float(lo)
                if b[0] <= la_f <= b[1] and b[2] <= lo_f <= b[3]:
                    lat, lon = la, lo
            except:
                pass
        owner = fac.get("owner", "")
        if "Ministry" in owner or "Government" in owner:
            ownership = "Public"
        elif "Private" in owner:
            ownership = "Private"
        elif any(k in owner for k in ("Faith","Church","Mission")):
            ownership = "Faith-Based"
        elif "NGO" in owner:
            ownership = "NGO"
        else:
            ownership = owner[:40]
        rows.append({
            "facility_name": fac["name"],
            "facility_code": fac.get("code", ""),
            "facility_type": fac.get("facility_type", ""),
            "facility_ownership": ownership,
            "latitude": lat, "longitude": lon,
            "admin1": fac.get("county", ""),
            "status": "Operating"
                if fac.get("status") == "Operational"
                else fac.get("status", ""),
        })
    with open(f"{OUTPUT_DIR}/kenya_standardized.csv", "w",
              encoding="utf-8", newline="") as f:
        w = csv.DictWriter(f, fieldnames=[
            "facility_name", "facility_code", "facility_type",
            "facility_ownership", "latitude", "longitude",
            "admin1", "status"])
        w.writeheader()
        w.writerows(rows)

    # Services CSV
    svc_rows = []
    for fac in all_facs.values():
        if fac["id"] in services:
            for svc in services[fac["id"]]:
                svc_rows.append({
                    "facility_code": fac.get("code", ""),
                    "service_category": svc["category"],
                    "service_group": svc["category"],
                    "service_detail": svc["name"],
                })
    with open(f"{OUTPUT_DIR}/kenya_services.csv", "w",
              encoding="utf-8", newline="") as f:
        w = csv.DictWriter(f, fieldnames=[
            "facility_code", "service_category",
            "service_group", "service_detail"])
        w.writeheader()
        w.writerows(svc_rows)

    wc = sum(1 for r in rows if r["latitude"])
    print(f"  Facilities: {len(rows)}")
    print(f"  With coords: {wc}")
    print(f"  Service records: {len(svc_rows)}")
    print(f"  Output: {OUTPUT_DIR}/")


# ══════════════════════════════════════════════
#  MAIN
# ══════════════════════════════════════════════

if __name__ == "__main__":
    print("KMHFR Extraction - HealthScope")
    print("Author: Kasim Chambulilo\n")
    t = time.time()
    all_facs = phase1()
    coords, services = phase2(all_facs)
    phase3(all_facs, coords, services)
    print(f"\nDone in {(time.time()-t)/60:.1f} minutes")
Note: In the script above, replace &lt;= with <= (less-than-or-equal) if copying from this HTML. The Python file version at scripts/kenya_extraction.py has the correct syntax.

7. Limitations & Notes

  • The KMHFR website does not have a documented public API; this extraction relies on parsing server-rendered HTML.
  • If the KMHFR updates its frontend framework or HTML structure, the regex patterns may need adjustment.
  • ~3.2% of facilities lack GPS coordinates in the KMHFR database.
  • Some service category names contain typos from the source (e.g., "INTERGRATED" instead of "INTEGRATED").
  • Coordinates are validated against Kenya's bounding box; out-of-bounds values are discarded.
  • The script makes ~18,000 HTTP requests total; please be respectful of the KMHFR server.
